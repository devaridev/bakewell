Maze Generator

by Devari (Rob Sykes)

This Maze Generator was built using Depth First Search algorithm. You can view the code here: https://github.com/devaridev/maze_generator

For the curious amongst you, this is built in TypeScript. It isn't using any libraries (except bootstrap for the visuals), just basic HTML5 Context Rects. Very lightweight.
